### Requirements
- gcc/clang
- cmake

## Optional
- doxygen (for docu not much at the moment)

### Building Library
cd <libpath> <br/>
mkdir build <br/>
cd build <br/>
cmake ../ <br/>
make <br/>
make install <br/>
